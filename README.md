# HR Analytics Dashboard (Tableau)

This project focuses on analyzing employee attrition and workforce trends
using an interactive Tableau dashboard.

## Key Insights
- Overall Attrition Rate: 15.26%
- Department-wise Attrition Analysis
- Age Group & Gender-wise Attrition
- Job Satisfaction Ratings

## Tools Used
- Tableau Public
- MS Excel / CSV
